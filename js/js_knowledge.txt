オブジェクト
　変数と関数の集合体
メソッド
　オブジェクトに関連付けされた関数のこと
プロパティ
　オブジェクトが持つ属性、情報
Elementオブジェクト
　HTMLドキュメント中にあるform要素内で配置されるエレメント（要素）を集めたオブジェクト
クッキー
　閲覧したWebサイトからスマホやPCの中に保存される情報
　サイトを訪れた日時や訪問回数など
カスタム属性
　html要素に対して設定することができる独自の属性
　属性名はdata-から始まる
JSON
　JavaScript Object Notation
　軽量のデータ交換フォーマット
　JavaScript構造データオブジェクトの表記法がベース
WebAPI
　HTTPプロトコルを用いてネットワーク越しに呼び出すアプリケーション間、システム間のインターフェース
Ajax
　Asynchronous JavaScript + XML
　JavaScriptとXMLを使って非同期にサーバとの間の通信を行うこと
SPA(SinglePageApplication)
　単一のWebページでアプリケーションを構成する設計構造
スクリプト言語
　習得するのを簡単にするために工夫されたプログラミング言語の総称
オブジェクト指向
　コンピュータプログラムの設計や実装についての考え方の一つで、互いに密接に関連するデータと手続き（処理手順）をオブジェクト（object）と呼ばれる一つのまとまりとして定義し、様々なオブジェクトを組み合わせて関連性や相互作用を記述していくことによりシステム全体を構築していく手法
コンパイラ言語
　人間が書いたプログラムをコンピュータが理解できる形式に変換するプログラム言語
　コンパイルする前にソースコードをオブジェクトコードに翻訳しておくため、プログラムを解釈しながら実行するインタープリタ型言語と比較して、パフォーマンスに優れている
JavaとJSの違い
　Java：
　　Sun Microsystems社が開発
　　一度プログラムを書けば、どんなハードウェアでも動作することを目的として開発
　　データ型を最初に固定で決める形式（静的型付）
　　コンパイルが必要
　　業務システムやTwitterなどのWebサービス、スマートフォン用（Android）向けアプリなどの開発に使用
　JavaScript
　　Netscape Communications社が開発
　　容易に開発できる手軽さと素早さを求めて開発
　　データ型を適宜変更できる形式（動的型付）
　　コンパイルが不要
　　Webブラウザ上で色を変えたり、ポップアップ表示をさせるといった動きがあるWebページなどの開発に使用
DOM
　Document Object Model
　HTMLドキュメントやXMLドキュメントをプログラムから利用するためのAPI
Node.js
　サーバーサイド用のJavaScriptアプリケーションのプラットフォーム
　JavaScriptをサーバーサイドやデスクトップで活用できる
